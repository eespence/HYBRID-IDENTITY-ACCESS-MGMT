## Featured Project — Hybrid Identity Deployment

Implemented a hybrid identity architecture integrating on-premises Active Directory with Microsoft Entra ID using a dedicated synchronization server.

The project includes:
- Domain controller configuration and validation
- Entra Connect synchronization
- Authentication failure investigation
- Root cause analysis (UPN namespace mismatch)
- Remediation and verification

This project demonstrates real-world identity engineering: diagnosing why authentication failed and restoring operational access.
